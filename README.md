# react-native-tink

tink sdk wrapper

## Installation

```sh
npm install react-native-tink
```

## Usage

```js
import { TinkView } from 'react-native-tink';

// ...

<TinkView clientId={' your client Id '} redirectUrl={' redirect url '}/>
```
